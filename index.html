Open your terminal or command prompt. 2. Create a new directory for your project and navigate into it: ```sh mkdir mynodeapp cd mynodeapp ``` #### Step 2: Initialize a Node.js Project 1. Initialize a new Node.js project with npm: ```sh npm init -y ``` This command will create a `package.json` file with default settings. #### Step 3: Create a Simple Node.js Application 1. Create an `app.js` file in the project directory: ```sh touch app.js ``` 2. Open the `app.js` file in your favorite text editor and add the following code: ```js const http = require('http'); const hostname = '127.0.0.1'; const port = 3000; const server = http.createServer((req, res) => { res.statusCode = 200; res.setHeader('Content-Type', 'text/plain'); res.end('Hello, World!\n'); }); server.listen(port, hostname, () => { console.log(`Server running at http://${hostname}:${port}/`); }); ``` This code creates a simple HTTP server that listens on port 3000 and responds with "Hello, World!" to any incoming requests. #### Step 4: Run the Application 1. In your terminal, run the application using Node.js: ```sh node app.js ``` 2. Open your web browser and navigate to `http://127.0.0.1:3000/`. You should see the message "Hello, World!". ### Explanation of the Code - **require('http'):** This line imports the built-in `http` module, which allows Node.js to transfer data over HTTP. - **http.createServer():** This method creates an HTTP server that listens to server ports and gives a response back to the client. - **server.listen():** This method makes the server listen on the specified hostname and port. - **Callback Function (req, res):** This function handles incoming requests (`req`) and responses (`res`). It sets the response status to 200 (success), sets the response header to `Content-Type: text/plain`, and ends the response with the message "Hello, World!".
